## Problem Statement
Train a drone agent to navigate safely under different weather 
conditions (calm, windy, storm) using Reinforcement Learning.

---

## Approach
- **Reinforcement Learning** — DQN-based agent training
- **Custom Environment** — Simulated weather conditions
- **OpenAI Gym** — Environment framework
- **PyTorch** — Model training

---

## Project Structure
| File | Description |
|------|-------------|
| `00_setup.ipynb` | Environment setup |
| `01_enviornment.ipynb` | Custom Gym environment |
| `02_agent.ipynb` | DQN agent design |
| `03_training.ipynb` | Model training (1000 episodes) |
| `04_testing.ipynb` | Testing in all weather conditions |
| `results/` | Trajectory plots and training curves |

---

## Results
- Trained for **1000 episodes**
- Tested in 3 weather conditions: **Calm, Windy, Storm**
- Generated trajectory and performance dashboards
